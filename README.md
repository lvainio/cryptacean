# Rust Cryptography 

This repository contains from scratch implementations of some crypto primitives. 

## Supported Hash Functions

- SHA0
- SHA1
- SHA224
- SHA256
- SHA384
- SHA512
- SHA512/224
- SHA512/256
- SHA3-224
- SHA3-256
- SHA3-384
- SHA3-512

- MD2
- MD4
- MD5

## Supported Encryption Functions

- RSA
